======================================================================================================

SISFO-TOKO v2.1 adalah sistem informasi manajemen Usaha Toko.
Hanya dibutuhkan sebuah WebServer (Apache+PHP+MySQL) untuk menjalankan aplikasi ini.


Konten - konten yang ada, mencakup :
	* Sistem Pembelian
	* Sistem Penjualan
	* Sistem Kasir
	* Sistem Stock Barang


Dengan pembagian akses, antara lain :
	* Administrator
	* Kasir/Pembuat Nota


Berikut beberapa contoh akses :
	admin
		user : admin
		pass : admin

	kasir/pembuat nota
		user : 1
		pass : 1




NB.
Untuk hasil terbaik, gunakan browser Firefox.
